## Git And GitHub

- `/root/velora/.git`가 빈 read-only 디렉터리로 잡혀 있어 일반 Git 명령이 실패하던 문제를 정리했다.
- 실제 Git 메타데이터는 `/root/velora/.git-real`에 있었고, `/root/velora/.git`를 다음 포인터 파일로 바꿨다.

```text
gitdir: .git-real
```

- 이후 일반 Git 명령이 정상 동작하는 것을 확인했다.
  - `git status`
  - `git remote -v`
  - `git log --oneline -1`

- 원격 저장소:

```text
git@github.com:smartcoresolution/velora_dev_server.git
```

- 이번 UI 작업 커밋:

```text
48b476d Improve senior-friendly frontend flow
```

- 브랜치:

```text
codex/dev-db-setup
```

- GitHub 원격에 푸시 완료:

```text
codex/dev-db-setup -> codex/dev-db-setup
```
